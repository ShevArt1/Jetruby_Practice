Класс 'Router' отвечает за маршрутизацию запросов к контроллеру 'PostsController'.

Метод 'initialize' создает пустой хэш '@routes', который будет хранить маршруты и соответствующие им действия контроллеров.

Метод 'init' вызывает метод 'resources' и запускает цикл взаимодействия с пользователем через консоль.
В цикле предлагает пользователю выбрать ресурс для взаимодействия. При выборе "1", вызывается метод 'connection' модуля 'Resource' для обработки запросов.

Метод 'resources' регистрирует маршруты для заданного контроллера и ключевого слова.
Создает новый экземпляр контроллера, заполняет хэш '@routes' маршрутами и соответствующими методами контроллера, выводит хэш маршрутов.


Модуль Resource
Содержит метод 'connection', который отвечает за обработку запросов:
Этот метод запускает цикл для взаимодействия с пользователем, где пользователь выбирает HTTP-метод и действие, после чего вызывает соответствующий метод контроллера.

'extend' используется для того, чтобы методы модуля 'Resource' стали методами класса, а не экземпляра класса. 
Это значит, что метод 'connection' будет доступен как метод класса 'PostsController', а не как метод его экземпляра.
